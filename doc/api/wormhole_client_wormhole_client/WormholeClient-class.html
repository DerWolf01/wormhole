WormholeClient

WormholeClient class

A client for managing connections and communications with a Wormhole server.

This class handles connecting to the server, sending and receiving messages, and managing the socket connection's lifecycle. It extends the SocketConnectionChangeNotifier to provide notifications on connection changes.
