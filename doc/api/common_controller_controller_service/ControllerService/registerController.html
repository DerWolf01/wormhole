Registers a controller with the service.

This method takes a controller instance, retrieves its path using the Controller annotation, and maps the path to the controller in the _controllerMap. This allows for the retrieval of controller instances based on their path.

Parameters:

  • controller: The controller instance to register.

Implementation

registerController(dynamic controller) {
  var path = Controller.byImplementation(controller).getPath;
  _controllerMap[path] = controller;
}
