ClientSession

ClientSession class

Represents a client session with a unique session ID and socket connection.

This class encapsulates the functionality for managing a client's session, including starting the session, receiving data, sending messages, and closing the session. It uses a Socket for communication and assigns a unique session ID based on the current timestamp.
