SerializableModel

SerializableModel class abstract

An abstract class that provides a foundation for models that can be serialized into JSON.

This class extends Model and includes a method for converting model instances into a JSON string. It is designed to be extended by more specific model classes that require JSON serialization capabilities. The toJson method leverages Dart's jsonEncode function to convert the model's data into a JSON string.
