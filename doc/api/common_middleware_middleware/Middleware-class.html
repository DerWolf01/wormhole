Middleware

Middleware<T extends SerializableModel> class

A class representing a middleware component.

This class allows for the creation and registration of middleware components within the application. Middleware components can perform actions before and after the main processing of a request.

Parameters:

  • path: The path on which the middleware should be applied.
  • preHandle: An optional callback to be executed before the main middleware logic.
  • postHandle: An optional callback to be executed after the main middleware logic.
