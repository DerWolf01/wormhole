ServerMessageService

ServerMessageService class

Provides server-side message handling services for the Wormhole application.

This service is responsible for processing incoming socket messages, both requests and responses, and dispatching them to the appropriate handlers. It extends SocketMessageService and implements the SocketMessageTypeRecognizer to facilitate this functionality.
